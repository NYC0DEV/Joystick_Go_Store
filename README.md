<h1>Joystick Go Store</h1>

<div class="box">
<p>
<b>Joystick Go Store</b> is an open-source community store for GameStation Go.  
It allows users to browse and download game and app binaries for their device directly from the store.
</p>
</div>

<div class="box">
<p>
Download <b>Joystick_Go_Store.zip</b>, extract it into  
<code>sd:\bin\Joystick_Go_Store</code>, then launch it.
</p>
</div>
