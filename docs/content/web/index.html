# Joystick Go Store **Joystick Go Store** is an open-source community store for GameStation Go. It allows users to browse and download game and app binaries. 📦 **[Open Releases Page](https://github.com/NYC0DEV/Joystick_Go_Store/releases)** --- ## Installation Download **Joystick_Go_Store.zip**, and extract it into: ```text sdcard/bin/Joystick_Go_Store/games ``` --- # 🎮 Controls | Button | Action | |---------|--------| | **D-Pad** | Navigate | | **A** | Select | | **B** | Back / Quit | | **START** | Launch application | --- # 📦 ZIP Package Requirements Applications uploaded to the Joystick Go Store must be packaged as a **.zip** archive containing a **Linux ARM32 (arm-linux-gnueabihf)** executable. The executable inside the archive **must have the exact same name as the .zip file** (without the `.zip` extension). ### Example ```text MyGame.zip ├── MyGame ├── assets/ ├── data/ └── config.json ``` When a user installs your application, the contents are extracted into a new folder within: ```text sdcard/bin/Joystick_Go_Store/games/ ``` The store launches the executable using: ```text sdcard/bin/Joystick_Go_Store/games/[FolderName]/[FileName] ``` --- # 🛠️ Build Instructions To compile and run the store application, ensure your build environment targets the ARM32 architecture and all required system headers are present. ## System Requirements - ARM32 Toolchain (`arm-linux-gnueabihf`) - SDL2 development libraries - FFmpeg - `libavformat` - `libavcodec` - `libavutil` - `libswresample` - `libswscale` - libcurl - nlohmann-json - Build tools (`gcc`, `make`, optional `cmake`) --- ## Install Dependencies (Ubuntu / Debian) ```bash sudo apt update && sudo apt install \ libsdl2-dev \ libavformat-dev \ libavcodec-dev \ libavutil-dev \ libswresample-dev \ libswscale-dev \ libcurl4-openssl-dev \ nlohmann-json3-dev \ build-essential ``` --- ## ARM32 Toolchain Download the ARM GNU Toolchain: https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads **Target** ```text arm-linux-gnueabihf (ARMv7 32-bit) ``` --- ## Compiling bash cd Joystick_Go_Store-main arm-linux-gnueabihf-g++ \ -O2 \ -std=c++17 \ store.cpp \ -o Joystick_Go_Store \ -lSDL2 \ -lavformat \ -lavcodec \ -lswresample \ -lavutil \ -lswscale \ -lpthread \ -ldl \ -lm ``` > **Notes** > > - `store.cpp` must be inside the project folder. > - Run the build command from the project root. --- ## License This project is open source.
